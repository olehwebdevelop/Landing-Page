# Apollo 11 Landing Page

Modern informational landing page inspired by the Apollo 11 Moon mission, designed with a cinematic space-themed interface and responsive layout.

## Features

- Fully responsive design
- Modern space-inspired UI
- Smooth scrolling and subtle animations
- Informational content sections
- Timeline-based layout elements
- Performance-focused frontend structure
- Clean typography and visual hierarchy

## Technologies

- HTML5
- CSS3
- JavaScript

## Sections

- Hero
- Mission Overview
- Launch Timeline
- Crew Information
- Moon Landing Details
- Gallery
- Contact / Footer

## Goals

This project was created to explore modern landing page design, storytelling through UI, and responsive frontend development while presenting historical information in an engaging visual format.
